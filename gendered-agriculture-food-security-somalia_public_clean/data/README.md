# Data

This folder contains only **derived aggregate outputs** used for public figures and communication.

It does not contain raw SIHBS household microdata, household-level records, personal identifiers, GPS coordinates, restricted files, or internal audit datasets.

## Included files

- `sample_flow.csv` — public sample-flow summary
- `key_insights_summary.csv` — concise public insight table
- `table_1_sample_profile_by_operator_sex.csv` — aggregate profile indicators
- `table_2_gendered_role_proxies.csv` — aggregate role-proxy indicators
- `table_3_crop_market_indicators.csv` — aggregate crop and market indicators
- `table_4_shocks_food_security.csv` — aggregate shock and food-security indicators
- `table_5_adjusted_associations_female_operator.csv` — model-summary estimates

No `example_inputs/` folder is included because the project is designed to run from the public aggregate outputs already provided here.
