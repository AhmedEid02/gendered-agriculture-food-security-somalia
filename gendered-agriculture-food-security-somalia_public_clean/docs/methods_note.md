# Methods note

This mini-project uses derived aggregate outputs from secondary household survey analysis.

## Analytical framing

The public-facing idea is role-sensitive gender analysis. Instead of relying only on household-head sex, the project focuses on the sex of the primary agricultural land operator and related role indicators.

## Public indicators

The public outputs summarise:

- agricultural land operation;
- agricultural work and land-right proxies;
- crop-category profiles;
- crop sale and hired labour use;
- climate-livelihood shock exposure;
- household food-security experience proxies;
- adjusted association estimates.

## Interpretation

The estimates shown here are associations, not causal effects. The public project is intended for communication and learning, not as a substitute for the full restricted-data analysis.
